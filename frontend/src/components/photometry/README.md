# Photometry Module Components

Этот модуль разбит на следующие компоненты для соблюдения лимита в 400 строк на файл.

## Структура компонентов:

### Константы и утилиты
- **pointDefinitions.js** ✓ - Определения точек для различных проекций
- **photometryConstants.js** - Константы измерений и норм
- **photometryUtils.js** - Утилитарные функции (расчеты расстояний, углов)

### Custom Hooks
- **usePhotometryState.js** - Управление состоянием модуля
- **usePhotometryCalculations.js** - Функции вычислений (измерения, углы)
- **useCanvasHandlers.js** - Обработчики событий canvas
- **useCanvasDrawing.js** - Функции отрисовки на canvas
- **useCalibration.js** - Логика калибровки

### UI Компоненты
- **PatientInfoPanel.js** ✓ - Панель с информацией о пациенте
- **ImageUploadPanel.js** - Панель загрузки изображений
- **PhotometryCanvas.js** - Canvas для отрисовки и работы с точками
- **CalibrationPanel.js** - Панель калибровки
- **ToolbarPanel.js** - Панель инструментов
- **PointsListPanel.js** - Список и управление точками
- **VisualizationPanel.js** - Настройки визуализации (плоскости, углы)
- **MeasurementsPanel.js** - Панель измерений
- **ReportPanel.js** - Панель отчетов и экспорта

### Главный компонент
- **PhotometryModuleNew.js** - Главный компонент, объединяющий все части

## Рекомендуемый порядок рефакторинга:

1. Создать файлы с константами и утилитами
2. Извлечь хуки (state, calculations, handlers)
3. Разбить JSX на под-компоненты
4. Обновить главный компонент для использования новых модулей

## Целевые размеры файлов:
- Константы: ~100-200 строк
- Хуки: ~300-400 строк
- UI компоненты: ~200-400 строк
- Главный компонент: ~300-350 строк
